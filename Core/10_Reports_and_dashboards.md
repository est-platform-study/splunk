## Reports
- reports can include time change picker
- at 'run as' setting, admin can create access limitation
- in report section, user can create report acceleration which make search faster by extract some data you want to see
- chart can be based on time, number, location
- user, power, admin can create reports

## Visualizations
- visualization can be saved as dashboard panel or reports

## Dashboards
- dashboard can be edited by source code
- time range picker is available if user want
- the time range picker will only work on panels with an inline search
- inline search is kinda search which is exist in xml code
- if report or dashboard panel is released at first, it doesn't include search 
- after change dashboard panel to include search which is inline search, dashboard panels can share time range picker
